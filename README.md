This repository contains a comprehensive, well-organized guide to essential Git commands, perfect for students and developers—from basics to advanced workflows. It's structured with clear examples, use cases, and quick references to help you master version control quickly.

Features
Beginner-friendly: Start with core commands like init, add, commit.

Branch mastery: Covers branch, checkout, merge, and rebase with conflict resolution.

Advanced tools: Includes stash, cherry-pick, revert, and reset with safety tips.

Practical examples: Real-world scenarios and pro tips for daily use.

Quick reference: Handy comparison table for common scenarios.
